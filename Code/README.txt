===METRICSTICS=== Version 1.0 2023/11/22

- This software is designed for the calculation and analysis of various descriptive
statistics related to software projects.

--------------------------------------------------------
Installation notes
--------------------------------------------------------

- Clone the METRICSTICS repository from GitHub:
git clone https://github.com/theOGCodeWitcher/SOEN-6611-METRICSTICS.git

- Navigate to the project directory:
cd METRICSTICS

--------------------------------------------------------
Usage
--------------------------------------------------------

- To obtain descriptive statistics for a dataset, run main.py file.
python main.py

- Follow the on-screen instructions to either input your dataset (.csv or .txt files only) or generate randomized data.

- You can view your data in the box provided.
